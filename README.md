This is the source code for the sole purpose of NTU EEE DIP EE012 Project ONLY.

To install and run this program, 
1) Download Android Studio from here: https://developer.android.com/studio
2) Download all the files in this Github Repository and put it in the folder.
3) Launch Android Studio and Create a NEW PROJECT
4) Open the folder and select the download folder.
5) Open Manifest.xml ensure that the project is sync.
   (If it ask to select Android SDK Development, copy and paste one of the implementation code to a line below or above it)
7) Connect your android device and ensure developer option is on
8) Under developer option -> Wired Debugging, make sure this function is on
9) Build the project
10) Run the code


Summary of the Codes:

KOTLIN FILE: All the logic and implementation of the layout

RES -> Layout: All the files that define the layouts constraints of our app

Python File: All the authentication modules implemented
